## Security

The Anthropic API key lives **only** in Supabase project secrets (`supabase secrets set ANTHROPIC_API_KEY=...`). It is read at runtime by `Deno.env.get('ANTHROPIC_API_KEY')` inside the edge function. The browser never sees it — it only calls `supabase.functions.invoke()` with the anon key. Keys must never go into `.env.local`, chat, or any file in the repo.

To rotate or update the key:
```
SUPABASE_ACCESS_TOKEN=<pat> npx supabase secrets set ANTHROPIC_API_KEY=<key> --project-ref gxszeczokcexmroklqvx
```

To redeploy the edge function after changes:
```
cd ~/Documents/Projects/clean-shopper && SUPABASE_ACCESS_TOKEN=<pat> npx supabase functions deploy chat --project-ref gxszeczokcexmroklqvx
```

## How context injection works

The edge function accepts an optional `context` string alongside `messages`. When present, it is appended to the system prompt:

```ts
const system = context ? `${SYSTEM_PROMPT}\n\n---\n\n${context}` : SYSTEM_PROMPT
```

The browser sends context via `supabase.functions.invoke('chat', { body: { messages, context } })`. Context is formatted plain text — no schema changes, no extra tables. Two formatters live in `src/lib/catalog.ts`:

- `formatCategoryContext` — up to 25 products with ingredients and certifications, ordered by clean score
- `formatProductContext` — single product with full numbered ingredient list

## Model

**`claude-haiku-4-5-20251001`** — fastest, lowest-cost Claude model. `max_tokens: 300`. Replies are intentionally short. If longer answers are needed later, raise `max_tokens` in `supabase/functions/chat/index.ts` and redeploy.
